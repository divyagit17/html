What does HTML stand for?HTML stands for Hyper Text Markup Language. It is the standard language used to create and design webpages.What is the purpose of <!DOCTYPE html>?The <!DOCTYPE html> declaration defines the document type and version of HTML.It tells the browser that the page is written in HTML5, which helps the browser render the content correctly.Define an HTML element with an example.An HTML element is a complete structure consisting of a start tag, content, and end tag.Example:htmlCopyEdit<p>This is a paragraph.</p>Here, <p> is the start tag, This is a paragraph. is the content, and </p> is the end tag.What is the use of the alt attribute in the <img> tag?The alt (alternative text) attribute provides a text description of the image if it cannot be displayed.It also improves accessibility for screen readers.Example:htmlCopyEdit<img src="cat.jpg" alt="A cute cat sitting on a couch">What is the difference between <p> and <br>?<p> defines a paragraph and adds spacing before and after the text block.<br> creates a line break and moves the following content to a new line without starting a new paragraph.Example:htmlCopyEdit<p>This is a paragraph.</p>This is line one.<br>This is line two.Write an example of a nested HTML element.Nested elements are elements placed inside other elements.Example:htmlCopyEdit<p>This is a <strong>bold</strong> word inside a paragraph.</p>What happens if you forget to close an HTML tag? Give an example.Forgetting to close a tag can lead to unexpected layout issues or cause the browser to interpret the HTML incorrectly.Example (Incorrect):htmlCopyEdit<p>This is a paragraphThis may cause the entire rest of the page to be treated as part of the paragraph.Write the syntax of an anchor (<a>) tag that opens a link in a new tab.htmlCopyEdit<a href="https://www.example.com" target="blank">Visit Example</a>The target="_blank" attribute tells the browser to open the link in a new tab or window
